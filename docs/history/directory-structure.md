
### Directory Structure of Agents

The following is the directory structure of the agents:

*   **poker_ai/**
    *   **__init__.py**: Initializes the poker_ai module.
    *   **agent.py**: Contains the Agent class, which is the base class for all agents.
    *   **card.py**: Contains the Card class, which represents a single card.
    *   **deck.py**: Contains the Deck class, which represents a deck of cards.
    *   **evaluator.py**: Contains the Evaluator class, which evaluates the strength of a hand.
    *   **hand.py**: Contains the Hand class, which represents a player's hand.
    *   **player.py**: Contains the Player class, which represents a player at the table.
    *   **table.py**: Contains the Table class, which represents the poker table.
